# User Roles SQL Schema
> ✅ Status: Completed

Relational database design project converting an ER diagram into SQL Server tables.

## Tables Created
| Table | Description |
|---|---|
| role | Stores user roles like Admin, Editor |
| user_account | Stores user login credentials |
| status | Stores status types |
| user_has_role | Links users to roles with timestamps |
| user_has_status | Links users to statuses with timestamps |

## Concepts Used
- Primary keys and Foreign keys
- Relational design from ER diagram
- INSERT and DELETE with constraint ordering

## Tools
SQL Server · Azure SQL Edge · Docker · VS Code
